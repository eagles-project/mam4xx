# Scripts

* `compare_bfb_hashes.sh`: compares BFB hashes between two mam4xx branch validation test runs.
* `compare_model_performance.sh`: compares the performance of two branches of E3SM running mam4xx.
* `compare_model_performance_plot.py`: used by `compare_model_performance` to plot output.
* `make_kernel_plots.sh`: plots kokkos-tools kernel profile results for an E3SM case (needs cleanup).
* `plot_kernel_times.py`: used by `make_kernel_plots.sh` to plot output.
* `quick_ne4pg2_cime_test_frontier`: launches a test run of the E3SM model with MAM4xx on Frontier.
